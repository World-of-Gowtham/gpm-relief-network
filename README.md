# GPM Relief Network 🛡️ | Tenkasi Disaster Management

A professional, real-time disaster response and NGO donation platform specifically designed to support the Tenkasi District. This project bridges the gap between those in danger, administrators, and volunteers.

## 🚀 Key Features

* **Live Weather Scanning:** Integrates with OpenWeather API to monitor Tenkasi's climate. The system automatically triggers "Caution" alerts if rain or thunderstorms are detected.
* **Emergency SOS Portal:** A dedicated reporting system that allows users to send instant disaster reports with location and severity details.
* **Instant Admin Alerts:** SOS reports trigger immediate, professional HTML-formatted email notifications to the admin via Google Apps Script.
* **Admin Help Broadcast:** A powerful feature allowing administrators to send mass email notifications to all registered users to request urgent help or mobilize volunteers during a crisis.
* **Verified NGO Donations:** Secure support system where donors can pick an NGO and provide UTR (Unique Transaction Reference) codes for instant verification.
* **GPM Relief Buddy:** An integrated AI chatbot that helps users navigate the site, check weather, and report emergencies.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (ES6+)
* **Backend:** Google Apps Script (Server-side logic & Emailing)
* **Database:** Google Sheets (Real-time data logging for SOS and Users)
* **APIs:** OpenWeatherMap API
* **Design:** Font Awesome Icons, Animate.css, Modern Dark Theme

## 📁 Project Structure

* `index.html`: Main dashboard with live weather, NGO cards, and chatbot.
* `report.html`: The user-facing emergency SOS reporting form.
* `admin.html`: Secure panel for weather monitoring and sending mass help requests.
* `script.gs`: The "Brain" of the project—handles all database logging and email triggers.
